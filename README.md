# Weather App

A simple web project that demonstrates the use of HTML, CSS, and JavaScript.

## Project Structure

- **index.html**: The main HTML file for the project.
- **style.css**: Contains the styles for the project.
- **script.js**: Contains the JavaScript logic for the project.
- **images/**: Directory for storing image assets.
- **apiKey.txt**: File for storing API keys (not included in the repository for security reasons).

## Prerequisites

To run this project, you need:

- A modern web browser (e.g., Chrome, Firefox, Edge).
- A code editor (optional, e.g., Visual Studio Code) if you want to modify the code.
- A Api key from Visual Crossing Web API
